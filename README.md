## Installation

You need to install ffmpeg

Download Brew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Then install ffmpeg
```
brew install ffmpeg
```
 
 ### Install all dependencies
 
 ```
  pip install -r requirements.txt
 ```
